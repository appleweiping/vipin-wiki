# vipin wiki

`vipin wiki` is a personal knowledge base designed to be maintained by an LLM agent.

It uses three layers:

- `raw/`: immutable source materials
- `wiki/`: LLM-authored markdown knowledge pages
- `AGENTS.md`: the operating manual that tells an agent how to maintain the wiki

Suggested workflow:

1. Put new source material into `raw/inbox/` or another folder under `raw/`.
2. Ask your agent to ingest the source into the wiki.
3. Read the updated pages in Obsidian.
4. Ask follow-up questions and file valuable answers back into `wiki/`.
